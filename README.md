# Evidence-driven-search

Local OSINT graph application with:

- graph-based UI for evidence visualization;
- background search jobs with live progress log;
- username, phone, email, domain, and image metadata flows;
- RU/global manual source packs;
- optional external engines: Sherlock, Maigret, Holehe.

## Run

```powershell
./start.bat
```

Or:

```powershell
./start.ps1
```
